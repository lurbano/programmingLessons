Quadratic Functions

Quadratics

Write a program to solve the following problems related to quadratic functions. To do these you are expected to be familiar with quadratics.

  • Find the zeros of a given quadratic. The zeros are the x values when the curve crosses the x-axis (i.e. when y = 0)
  • Find the x,y coordinates of the minimum or maximum of the function (the critical value).
  • Determine if the critical value is a maximum or a minimum.
  • For a given x value (say x = 2), find the slope of a tangent to the quadratic.
  • Given two x values, (say x = -2 and x = 2)
    • find the length of a chord between the two points on the curve.
    • approximate the length of the curve between two points by using a series of chords.

For testing, use the equation: $$ y = x^2 - 2x - 3 \tag{1}$$
