Lists

Make a list when you want to store multiple items as a single variable.

In this exercise, we'll use lists to keep track of students' grades.

StudentExam 1Exam 2
Brooke 85 95
Coen 90 100
Blas 80 100
George 50 96

Notes and Code
Outcome

I. Creating Lists

We'll make three lists. One for the student names, and one each for the two exams. 


                students = ['Brooke', 'Coen', 'Blas', 'George']
                exam1 = [85, 90, 80, 50]
                exam2 = [ 95, 100, 100, 96]

Observe:

  • Lists are delimited using square brackets: []
  • Items in the list are separated by commas (,)
  • The students list is a list of four strings (words) delimited by quotation marks (you can use single or double quotation marks)
  • The exam lists are lists of numbers (hence the lack of quotation marks)

If we'd like to print out the whole list we can just write: 


                  print(students)
which gives, 

                  ['Brooke', 'Coen', 'Blas', 'George']

However, if we want to print out just one of the items in the list, we have to reference it by its index. Indexing starts with 0 so to get the first student's name we'd use: 


                  print(students[0])
which gives, 

                  Brooke

II. Referencing Items in a List

  • Forward Indexing:
    • Starts at 0 for the first item.
      • students[0] --> 'Brooke'
      • students[1] --> 'Coen'
  • Backward Indexing (in python, but not all languages):
    • The last item in the list has the index -1
      • students[-1] --> 'George'
      • students[-2] --> 'Blas'
  • Slicing: To get a number of items from the sequence give the two indices separated by a colon (:):
    • The
      • students[1:3] --> ['Coen', 'Blas']
      • students[:2] --> ['Brooke', 'Coen']
      • students[1:] --> ['Coen', 'Blas', 'George']
      • Notice that slicing gives the results as a shorter list.

So the program: 

                students = ['Brooke', 'Coen', 'Blas', 'George']
                print('students[0]:', students[0])
                print('students[1]:', students[1])
                print('students[-1]:', students[-1])
                print('students[-2]:', students[-2])
                print('students[1:3]:', students[1:3])
                print('students[:2]:', students[:2])
                print('students[1:]:', students[1:])

Outputs:


                students[0]: Brooke
                students[1]: Coen
                students[-1]: George
                students[-2]: Blas
                students[1:3]: ['Coen', 'Blas']
                students[:2]: ['Brooke', 'Coen']
                students[1:]: ['Coen', 'Blas', 'George']
