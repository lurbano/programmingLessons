Setup LEDs (Hardware and Software)

References:
  • Python Lists Documentation.

Hardware

You will need:

  1. Raspberry Pi Zero W (should also work with the B). The setup assumes that the Pi has a wifi chip.
    1. The Pi Zero WH, which includes headers would be the easiest to start with. Otherwise, you'll need to solder on your own.
    2. I've also used the Pi Zero 2 W.
  2. WS281x LED strips (tested with WS2812B). These are individually addressable strips. The default programs use 20 LEDs.
    1. If you have a month of lead time, I've used these before.
  3. Jumper cables
    1. If you have a Pi with headers, and a new LED strip, you should be able to connect everything just using male-to-male jumpers.

Software

rpi-led-strip: Software setup instructions are detailed at the GitHub repository.

  1. Assumes that the led strip is controlled by the GPIO 18 pin.
