LEDs (Controlling using Python)

Learn to program using LED strips controlled by a Raspberry Pi.
Setting up the Hardware and Software on the Pi
  1. Setup rpi-led-strip (For Pi computers: Zero, B etc.)
  2. Setup ledPixelsPico (For Pi Pico microcontroller board)
Assemble hardware (Raspberry Pi and LED strip), install the operating system on the Pi. For the Pi Zero (or B's), connect Pi to the local wireless network. For the Pico, you just plug it in to the USB.
[For Pi computers (Zero, B's)]: Connecting to the Pi and navigating using the command line.
Connecting using the command line.
Connect to the Raspberry Pi remotely, use the command line to navigate the file system, and run python programs that control the leds.
Lessons on learning to code using python on the Pi and a LED strip.
